# Portfolio de développeuse web – Projet OpenClassrooms

Ce projet est réalisé dans le cadre du parcours **Développeur Web** chez OpenClassrooms.  
L’objectif est de concevoir, développer et déployer un **portfolio professionnel** afin de présenter mes projets, mes compétences et mon parcours de développeuse web.

---

## Fonctionnalités

- Présentation du profil et du parcours professionnel
- Mise en valeur des projets réalisés durant la formation
- Modales projets avec descriptions détaillées
- Architecture en composants avec React
- Design responsive avec Tailwind CSS
- Navigation fluide et intuitive
- Optimisation du référencement (SEO)
- Prise en compte de l’accessibilité
- Déploiement du site en ligne

---

## Technologies utilisées

- **React**
- **Tailwind CSS**
- **JavaScript**
- **Git & GitHub**
- **Vercel**
- **Visual Studio Code**
